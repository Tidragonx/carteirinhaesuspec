Política de Privacidade - Gerador Carteirinha SUS e-SUS

O Gerador Carteirinha SUS e-SUS foi desenvolvido para gerar e imprimir a carteirinha do SUS diretamente na página do e-SUS, usando os dados já exibidos no cadastro do cidadão.

Dados acessados

A extensão pode acessar dados visíveis na página do e-SUS, incluindo informações cadastrais como nome, CPF, CNS, data de nascimento, sexo e outros dados necessários para preencher a carteirinha.

Forma de uso dos dados

Os dados são utilizados exclusivamente para montar a pré-visualização e a impressão da carteirinha do SUS dentro do navegador.

Compartilhamento de dados

A extensão não vende, não compartilha e não transmite esses dados para servidores externos.

Armazenamento

A extensão não mantém banco de dados próprio e não realiza armazenamento externo das informações acessadas, exceto pelo processamento local necessário para gerar a carteirinha durante o uso.

Escopo de funcionamento

O funcionamento da extensão é restrito ao domínio autorizado do e-SUS configurado no manifesto da extensão.

Segurança

O processamento dos dados ocorre localmente no navegador, com o único objetivo de montar o modelo da carteirinha para conferência e impressão.

Alterações nesta política

Esta Política de Privacidade pode ser atualizada futuramente para refletir melhorias ou mudanças funcionais da extensão.
